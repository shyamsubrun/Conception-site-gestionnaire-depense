# Gestionnaire de Dépenses - Application Web (React)

## Description

Ce projet consiste en une application web de gestion de dépenses, développée en utilisant React pour le frontend. L'application permet aux utilisateurs d'inscrire leurs dépenses, de les catégoriser, et de visualiser graphiquement leurs dépenses sur plusieurs mois.

### Fonctionnalités

1. **Inscription des Dépenses**
   - Ajout de nouvelles dépenses avec des détails tels que le montant, la date et la catégorie.
   - Catégorisation des dépenses pour une meilleure organisation.

2. **Visualisation Graphique des Dépenses**
   - Affichage graphique des dépenses sur une période donnée (par mois).
   - Graphiques intuitifs permettant de comprendre les tendances de dépenses.

### Technologies Utilisées

- **React.js** : Bibliothèque JavaScript pour le frontend.
- **Chart.js** : Bibliothèque pour la création de graphiques.
- **CSS** : Style et mise en page.

### Installation

1. Clonez le dépôt : `git clone https://github.com/votre-utilisateur/ProjetGestionDepensesReact.git`
2. Installez les dépendances : `npm install` dans le répertoire du projet React.
3. Démarrez l'application : `npm start` pour lancer le serveur de développement.

### Structure de Données

La gestion des données peut être simplifiée avec une structure basique :
- **dépenses** : Table comprenant des champs tels que l'id, le montant, la date, la catégorie, etc.

### Remarques

- Ce projet utilise React pour offrir une expérience utilisateur interactive.
- La visualisation graphique des dépenses est réalisée avec la bibliothèque Chart.js.
- Les utilisateurs peuvent inscrire leurs dépenses, les catégoriser et visualiser graphiquement leurs dépenses sur plusieurs mois.
